Esse projeto foi criado com a intençao de ser nosso primeiro projeto em Kotlin para as aulas do 3° Ano de SI na FIAP. Nesse repositório contém todo nosso aprendizado até agora, desde a criaçao do projeto, limpeza da MainActivity, compreensão de estruturas básicas do código, implementaçao do jetpack compose, criaçao de chamadas e a própria navegaçao entre telas via NavCompose. Abaixo segue imagens sobre meu primeiro projeto funcional de navegaçao entre telas simples.
